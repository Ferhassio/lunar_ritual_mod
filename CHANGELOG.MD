## [0.1.0] - Initial Release

### Added
- Implemented a functioning system for the unique persistent currency **Primordial Shards**
- Primordial Shards can **drop during gameplay** and are **saved between runs** [`100% drop chanse for Beta test`]
- Added a **Ritual System** available at the start of each run
- Added **8 unique rituals**
- Each ritual has **3 offering tiers** (1, 5, and 10 shards)
- Multiplayer compatibility
- Alloyed Collective compatibility
